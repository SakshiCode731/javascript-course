# JavaScript Tutorial Collection

A collection of JavaScript tutorial files covering fundamental programming concepts in JavaScript.

## About This Project

This repository contains educational JavaScript files that demonstrate various JavaScript concepts including:

- Variables and constants
- Primitive data types
- Operations and expressions
- Conditional statements
- Functions and more

## How to Use

When you run this project in Replit, an interactive menu will appear in the console. Simply:

1. Choose a tutorial number (1-7) from the menu
2. Press Enter to run that specific tutorial
3. View the output and learn from the examples

### Available Tutorials

1. **Basic Console & Functions** - Introduction to console output and functions
2. **Variables** - Understanding variables in JavaScript
3. **var, let & const** - Differences between var, let, and const
4. **Primitive Datatypes** - Working with primitive data types
5. **Practice Set** - Practice exercises combining concepts
6. **Operations & Expressions** - Arithmetic, comparison, and logical operators
7. **Conditional Expressions** - If statements, switch cases, and ternary operators

## Running Tutorials

The project automatically starts with an interactive menu. You can also run individual tutorials directly:

```bash
node tut1.js
node "tut2(variable).js"
node "tut5(practice_set).js"
```

## Dependencies

- **Node.js 20** - JavaScript runtime
- **prompt-sync** - For interactive console input (used in tutorial 7)

All dependencies are automatically managed through the package.json file.
