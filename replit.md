# JavaScript Tutorial Collection

## Overview
This is a collection of JavaScript tutorial files covering fundamental concepts. The project provides an interactive menu system to run different tutorial examples.

## Project Structure
- **index.js** - Main entry point with interactive menu to select and run tutorials
- **tut1.js** - Basic console output and functions
- **tut2(variable).js** - Variables in JavaScript
- **tut3(var_let and _const).js** - Understanding var, let, and const
- **tut4(primitive-datatype).js** - Primitive data types
- **tut5(practice_set).js** - Practice exercises
- **tut6(Operation_&_Expression).js** - Operations and expressions
- **tut7_Conditional_Expression.js.js** - Conditional expressions and switch statements
- **html_file/** - Directory for HTML files (currently empty)

## Dependencies
- **Node.js 20** - Runtime environment
- **prompt-sync** - For interactive console input

## How to Run
The project is set up with a workflow that automatically runs `node index.js`. This presents an interactive menu where you can select which tutorial to run (1-7) or exit (0).
